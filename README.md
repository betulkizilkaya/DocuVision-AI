# DocuVision AI 🤖📄

DocuVision AI is a **multi-modal artificial intelligence platform** designed for  
**document intelligence** and **computer vision** applications 🗂️.

The system enables automated **text & image extraction from documents**,  
**text and image similarity analysis**, and  **document & image classification**,  
with a scalable architecture that will be extended to **vehicle and license plate recognition** 🚗.

## ✨ Key Features
- Automated text & image extraction from PDF documents  
- Text similarity analysis using NLP techniques  
- Image similarity analysis using computer vision methods  
- Document & image classification pipelines  
- Modular & scalable architecture for future vision-based extensions  

## 🛠️🌐 Technologies
- **Python**  
- **NLP:** TF-IDF, Word2Vec, Cosine Similarity  
- **Computer Vision:** OpenCV, ORB, SSIM  
- **Machine Learning:** SVM, traditional ML classifiers  
- **Backend:** Flask, SQLite
